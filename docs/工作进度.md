# 危险标志识别系统 — 工作进度

## 项目概况

基于深度学习的危险标志识别系统，采用 YOLOv8 目标检测模型，支持图片/视频/摄像头实时检测。

**当前阶段：系统开发完成，待测试与撰写设计报告。**

---

## 已完成工作

### 1. 数据采集

| 类别 | 英文名 | AI生成 | 网图 | 合计 |
|------|--------|:------:|:----:|:----:|
| 注意安全 | danger | 84 | 9 | 93 |
| 当心触电 | electricity | 88 | 10 | 98 |
| 当心火灾 | fire | 85 | 6 | 91 |
| 当心电离辐射 | ionizing_radiation | 90 | 10 | 100 |
| 当心火车 | train | 88 | 8 | 96 |
| **合计** | | **435** | **43** | **478** |

### 2. 数据预处理

| 步骤 | 脚本 | 功能 | 状态 |
|------|------|------|:--:|
| Step 1 | `train/prepare_data.py` | 缩放 640×640 + 重命名 + AI比例统计 | ✅ |
| Step 2 | makesense.ai | 人工 YOLO 标注 | ✅ |
| Step 3 | `train/split_dataset.py` | 校验 + 划分 7:2:1 + dataset.yaml | ✅ |

### 3. 模型训练

| 项目 | 详情 |
|------|------|
| 模型 | YOLOv8n (COCO 预训练) |
| epochs | 200 (epoch 34 早停) |
| batch / imgsz | 16 / 640 |
| freeze / optimizer | 5 层 / AdamW |

### 4. 模型评估

**Test 集:**

| mAP@0.5 | mAP@0.5:0.95 | Precision | Recall |
|:-------:|:------------:|:---------:|:------:|
| **0.983** | **0.768** | 0.947 | 0.993 |

所有 5 类 mAP@0.5 > 0.93。

### 5. 模型封装

| 文件 | 功能 |
|------|------|
| `src/models/config.py` | 类别/阈值常量 |
| `src/models/detector.py` | `SignDetector` — 支持 ndarray/文件路径输入 |
| `src/models/best.pt` | 训练产出权重 (6.3MB) |

### 6. 框架层

| 文件 | 功能 |
|------|------|
| `src/framework/video_processor.py` | 视频/摄像头帧读取, 迭代器 |
| `src/framework/temporal_filter.py` | 时空一致性校验 (双重阈值 + IoU 追踪) |

### 7. 后端 & 前端

| 端点 | 方法 | 功能 |
|------|------|------|
| `/api/register` | POST | 用户注册 (JSON 文件持久化) |
| `/api/login` | POST | 登录 (SHA-256 + salt) |
| `/api/reset-password` | POST | 密码重置 (演示版) |
| `/api/detect/image` | POST | 图片检测 → 标注图 + 结构化卡片 |
| `/api/detect/video` | POST | 视频检测 → Canvas 帧播放器 + 统计 |
| `/api/detect/live` | WS | 摄像头实时帧 → 标注帧回传 |

前端: `login.html` (登录/注册/重置) + `detect.html` (图片/视频/摄像头三 tab), 暗色 GitHub 风格主题。
