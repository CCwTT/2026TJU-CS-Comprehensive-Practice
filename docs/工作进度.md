# 危险标志识别系统 — 工作进度

## 项目概况

基于深度学习的危险标志识别系统，采用 YOLOv8 目标检测模型，支持图片/视频/摄像头实时检测。

**当前阶段：模型训练完成，待框架层与系统开发。**

---

## 已完成工作

### 1. 数据采集

| 类别 | 英文名 | AI生成 | 网图 | 合计 |
|------|--------|:------:|:----:|:----:|
| 注意安全 | danger | 84 | 9 | 93 |
| 当心触电 | electricity | 88 | 10 | 98 |

- AI 生成比例约 91%
- 数据来源：AI 生成 + 网络爬取

### 2. 数据预处理

| 步骤 | 脚本 | 功能 | 状态 |
|------|------|------|:--:|
| Step 1 | `train/prepare_data.py` | 缩放 640×640 (letterbox) + 去 alpha + 重命名 + 统计 | ✅ |
| Step 2 | 人工标注 | makesense.ai 标注 YOLO 格式 bbox | ✅ |
| Step 3 | `train/split_dataset.py` | 标注校验 + 数据集划分 7:2:1 + 生成 dataset.yaml | ✅ |

- **输出目录**：`data/training/`（train 333 / val 94 / test 51）

### 3. 模型训练

| 项目 | 详情 |
|------|------|
| 模型 | YOLOv8n（预训练 COCO 权重） |
| 训练脚本 | `train/train.py` |
| epochs | 200（实际 epoch 34 早停） |
| batch size | 16 |
| 输入尺寸 | 640×640 |
| freeze | 5 层 |
| 优化器 | AdamW (auto) |
